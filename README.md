# Sharely 📱

Sharely is a modern social media application built with Flutter that empowers users to share their opinions freely, engage in meaningful conversations, and connect with others.

## ✨ Features

*   **User System**: Secure authentication and seamless user profile management.
*   **Post Management**: Create, read, update, and delete your thoughts and opinions easily.
*   **Comment System**: Engage in discussions by leaving comments on posts.
*   **Follow System**: Follow your favorite users and build a personalized timeline/feed.
*   **AI Polish Feature**: Enhance your thoughts before sharing! Use the built-in AI tool to polish your text, refine grammar, and improve the tone of your posts automatically.

## 🏗 Architecture

This project strictly follows the **MVVM (Model-View-ViewModel)** architectural pattern to ensure a clean separation of concerns, scalability, and maintainability.

*   **Model**: Represents the core data and business logic (Entities, Repositories, API Services/Data Sources).
*   **View**: The UI layer built with Flutter widgets. It passively displays data and captures user inputs.
*   **ViewModel**: Acts as the bridge between the View and the Model. It handles presentation logic, state management, and updates the View when the Model changes.
